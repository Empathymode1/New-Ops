INTEGRATION POINTS
───────────────────────────────────────────────────────────────────────────────

ServiceClient Methods Used:
  • addJobListListener() - updates job lists
  • addJobStateListener() - updates individual job state
  • addEventListener() - receives transfer events
  • addConnectListener() - refresh on reconnect
  • startAll() / stopAll() - action buttons
  • getConfiguration() - settings refresh
  • sendUpdateConfig() - settings save

WebSocket Events Used:
  • JOB_STATE - per-job updates (status, heartbeat, etc.)
  • FILE_TRANSFERRED - activity feed
  • SERVICE_STARTED / SERVICE_STOPPED - state changes
  • ERROR events - error tracking
