## Key Changes

### Before (Inline Styles)
```java
nameLabel.setStyle(
    "-fx-font-size: 15;" +
    "-fx-font-weight: bold;" +
    "-fx-text-fill: " + Theme.TEXT_PRIMARY + ";");
```

### After (CSS Classes)
```java
nameLabel.getStyleClass().add("job-detail-header");
```

### CSS
```css
.job-detail-header {
    -fx-font-size: 15;
    -fx-font-weight: bold;
    -fx-text-fill: -fw-text-primary;
}
```

**Light Theme (Default):**
- Light beige/white backgrounds

**Dark Theme:**
