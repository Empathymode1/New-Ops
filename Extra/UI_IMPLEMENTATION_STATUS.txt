TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

COMPILATION ERRORS:

  "Cannot find symbol: Theme"
  → Ensure Theme.java is in same package (com.filewatcherui.ui)
  → Run: mvn clean compile

  "Cannot find symbol: ServiceClient"
  → Check imports in component files
  → Ensure filewatcher-ui can access ServiceClient

  "JavaFX not found"
  → Verify JavaFX 21 is in Maven dependencies
  → Check pom.xml for javafx-controls, javafx-fxml

RUNTIME ERRORS:

  "NullPointerException in WebSocket listener"
  → Verify client is connected before data updates
  → Check Platform.runLater() is used in all listeners

  "ToastNotification not appearing"
  → Ensure .show(stage) is called with valid Stage
  → Check if stage is still active

  "Keyboard shortcuts not working"
  → Verify KeyboardShortcuts is registered on Scene
  → Check Scene has focus

  "StatusBadge colors wrong"
  → Verify Theme constants match job status values
  → Check WatchJob.Status enum hasn't changed
